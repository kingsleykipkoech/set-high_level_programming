# Python - More Classes and Objects

This project explores advanced Object-Oriented Programming (OOP) concepts in Python using the `Rectangle` class as a progressive case study.

## Learning Objectives

- Class and instance attributes
- Properties, getters, and setters
- `__str__`, `__repr__`, and `__del__` special methods
- Class methods and static methods
- Instance counting with class attributes
- Configurable string representation
- The N Queens backtracking algorithm
