# Python - Classes and Objects

This project introduces Object Oriented Programming (OOP) in Python. It covers classes, objects, instances, attributes (public and private), methods, properties with getters and setters, data validation, special methods (`__init__`, `__str__`, and rich comparison methods), and building a data structure (singly linked list) with classes.

## Files

| File | Description |
|---|---|
| `0-square.py` | Empty class `Square` that defines a square |
| `1-square.py` | Class `Square` with a private instance attribute `size` |
| `2-square.py` | Class `Square` with size validation (type and value) |
| `3-square.py` | Class `Square` with a public instance method `area()` |
| `4-square.py` | Class `Square` with a property getter and setter for `size` |
| `5-square.py` | Class `Square` with a `my_print()` method that prints the square with `#` |
| `6-square.py` | Class `Square` with a `position` attribute used when printing |
| `100-singly_linked_list.py` | Classes `Node` and `SinglyLinkedList` implementing a sorted singly linked list |
| `101-square.py` | Class `Square` printable directly with `print()` via `__str__` |
| `102-square.py` | Class `Square` comparable with `==`, `!=`, `>`, `>=`, `<`, `<=` based on area |
| `103-magic_class.py` | Class `MagicClass` reconstructed from Python bytecode |

## Requirements

- All files interpreted/compiled on Ubuntu with `python3`
- All files start with `#!/usr/bin/python3`
- Code follows `pycodestyle` style guidelines
- All files are executable
